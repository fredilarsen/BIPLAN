

## Constants

| Constant   | Purpose                     |
| ---------- | --------------------------- |
| `false`    | or `0`                      |
| `true`     | or `1`                      |
| `LOW`      | or `0` digital pin state    |
| `HIGH`     | or `1` digital pin state    |
| `INPUT`    | or `0` digital pin mode     |
| `OUTPUT`   | or `1` digital pin mode     |
